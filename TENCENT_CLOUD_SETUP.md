## 目标
开启 CloudBase 匿名登录，设置 repairTickets 数据库权限

## 第一步：登录腾讯云控制台

1. 打开浏览器，访问：
   ```
   https://console.cloud.tencent.com/tcb/auth?envId=dorm-system-5g5k2k8k-1byd118ff42
   ```

2. 使用子账号登录：
   - 用户名：`dorm-system-5g5k2k8k-1byd118ff42@100046724893`
   - 密码：`~rjf,4Er`

---

## 第二步：开启匿名登录

1. 登录后，你应该在云开发控制台的「登录授权」页面
   
2. 找到「匿名登录」选项卡
   
3. 点击开关，将其设置为「**开启**」状态
   
4. 页面会显示「匿名登录已启用」

   💡 **注意**：开启后可能需要等待 1-2 分钟生效

---

## 第三步：设置数据库权限

1. 打开新标签页，访问：
   ```
   https://console.cloud.tencent.com/tcb/database/collections?envId=dorm-system-5g5k2k8k-1byd118ff42
   ```

2. 在集合列表中找到 `repairTickets`，点击进入

3. 点击顶部「权限设置」按钮

4. 在权限设置页面，选择：
   ```
   所有用户可读，仅创建者可写
   ```

5. 点击「保存」按钮

   💡 **说明**：
   - 「所有用户可读」= 任何人都能查看工单
   - 「仅创建者可写」= 只有提交工单的人能修改自己的工单
   - 管理员通过其他方式（如控制台）可以修改所有数据

---

## 第四步：验证设置

1. 等待 2-3 分钟让设置生效

2. 打开诊断页面测试：
   ```
   https://dorm-system-5g5k2k8k-1byd118ff42-1408974183.tcloudbaseapp.com/diagnose.html
   ```

3. 点击「开始诊断」按钮

4. 如果显示「所有测试通过」，说明设置成功！

## 常见问题

### Q1: 找不到「匿名登录」选项
**解决**：确保你在云开发控制台，不是主账号控制台。链接应该是：
```
https://console.cloud.tencent.com/tcb/...
```

### Q2: 权限设置后仍然无法访问
**解决**：
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 刷新页面重试
3. 等待 5 分钟再试（权限同步需要时间）

### Q3: 提示「权限不足」
**解决**：子账号可能没有足够权限，需要主账号操作，或联系管理员
